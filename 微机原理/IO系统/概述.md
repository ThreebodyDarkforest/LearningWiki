# 概述

## 输入输出接口

- 分类
	- 通用性：专用、通用
	- 传输模式：并行、串行
	- 可编程、不可编程
	- 时序控制：异步、同步
- 结构
	- IO端口：能被CPU直接访问的寄存器地址
		- 数据端口
		- 状态端口
		- 控制端口
	- 地址译码器：对地址译码，选择寄存器
	- 控制逻辑：接收CPU的命令
- 指令：高于8位的IO地址需要用DX传入
	- `IN AL/AX, 8位地址/DX`，读一个字节或一个字
	- `OUT 8位地址/DX, AL/AX`

## 中断方式

- 中断评优方式
	- 软件查询
	- 硬件判优：写死的电路
	- 专用控制器：芯片
